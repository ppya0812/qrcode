#qrcode   生成二维码

* 支持生成简单二维码

* 支持生成中间带logo的二维码

| 参数        | 说明        | 类型        | 可选值      |   默认值   |
|------------|-------------|------------|------------|-----------|
| text       | 文本         |  String    |            |           |
| image      | 中间logo地址（注意不要跨域）  | String     |            |           |
| size       | 二维码尺寸    | String     |            |  64       |
